## Quick start

1. Download `CLAUDE_TEMPLATE.md` and save it as `CLAUDE.md` in your project folder
2. Open Claude (or any AI coding assistant) pointed at that folder
3. Follow Steps 1–5 in the guide to set up your project
4. Repeat the build loop: **Open → Build → Run → Test → Close**
